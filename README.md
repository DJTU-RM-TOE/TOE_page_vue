## vuepress的安装
### 环境安装
- 由于要求nodejs18版本，添加[官网源](https://nodejs.org/zh-cn/download/package-manager)

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.0/install.sh | bash
```
```
nvm install 20
```
```
node -v
```
### 依赖安装

npm install slimsearch
